# Writeup Archive

Arsip writeup tim CTF kami yang diorganisir berdasarkan event.

## Struktur Repository

Repository ini berisi writeup dari berbagai kompetisi CTF yang telah diikuti oleh tim kami. Setiap event memiliki folder tersendiri dengan format:

```
events/
├── [Nama-Event-YYYY]/
│   ├── README.md          # Informasi umum tentang event
│   ├── challenge-1.md     # Writeup untuk challenge 1
│   ├── challenge-2.md     # Writeup untuk challenge 2
│   └── assets/            # File pendukung (gambar, script, dll)
```

## Format Writeup

Setiap writeup ditulis dalam format Markdown (.md) dan mencakup:
- Nama dan kategori challenge
- Deskripsi challenge
- Langkah-langkah penyelesaian
- Flag yang diperoleh
- Tools yang digunakan
- Lessons learned

**Catatan**: File PDF dapat disimpan di folder `assets/` sebagai dokumentasi tambahan jika diperlukan.

## Cara Berkontribusi

1. Buat folder baru untuk event dengan format `[Nama-Event-YYYY]`
2. Tambahkan file README.md di folder event dengan informasi:
   - Nama lengkap event
   - Tanggal pelaksanaan
   - Link ke website event
   - Peringkat tim (jika ada)
3. Tambahkan writeup untuk setiap challenge yang berhasil diselesaikan
4. Commit dan push perubahan
