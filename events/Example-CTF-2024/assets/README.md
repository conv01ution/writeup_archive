# Assets Folder

Folder ini berisi file-file pendukung untuk writeup, seperti:

- Screenshot dari challenge
- Script exploit yang digunakan
- File-file yang di-download dari challenge
- Diagram atau gambar penjelasan

## Naming Convention

Gunakan nama file yang deskriptif:
- `screenshot-login-page.png`
- `exploit-script.py`
- `challenge-binary`
- `network-capture.pcap`
